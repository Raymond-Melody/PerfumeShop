#### 1.4 浏览器缓存
**文件**：`web.config` (+3行)


**效果**：
- 静态资源缓存7天
- 减少HTTP请求

##  技术架构

### 懒加载实现
```javascript
// IntersectionObserver配置
var config = {
    rootMargin: '200px 0px',  // 提前200px加载
    threshold: 0.01           // 1%可见时触发
};

// HTML结构
<img data-src="真实图片.jpg" 
     src="占位图.svg" 
     class="lazy-placeholder">

// CSS动画
img[data-src] { opacity: 0; }
img[data-src].loaded { opacity: 1; transition: opacity 0.3s; }
```

### 暗黑模式实现
```javascript
// 主题优先级
1. localStorage用户选择
2. 系统偏好（prefers-color-scheme）
3. 默认亮色

// CSS变量
[data-theme="dark"] {
    --bg-color: #1A1A1A;
    --text-color: #E0E0E0;
    --card-bg: #2D2D2D;
}

// 系统监听
window.matchMedia('(prefers-color-scheme: dark)')
    .addEventListener('change', callback);
```

### GZIP压缩配置
```xml
<!-- web.config -->
<urlCompression 
    doStaticCompression="true"   <!-- CSS/JS/HTML -->
    doDynamicCompression="true"  <!-- ASP动态页面 -->
/>
```

##  使用指南

### 懒加载使用
```asp
<!-- 产品列表页（自动应用） -->
<img data-src="<%= ImageURL %>" 
     src="/images/default-product.svg" 
     class="lazy-placeholder">

```

### 主题切换使用
```javascript
// 手动设置主题
window.setTheme('dark');  // 或 'light'

// 获取当前主题
var current = window.getTheme();

// 切换主题
window.toggleTheme();
```

### WebP转换使用
```powershell
# 基本用法（转换images目录）
.\tools\convert_to_webp.ps1

# 指定目录和质量
.\tools\convert_to_webp.ps1 -SourceDir ".\images\products" -Quality 85

# 查看帮助
.\tools\convert_to_webp.ps1 -Help
```

### HTML中使用WebP
```html
<picture>
    <source srcset="image.webp" type="image/webp">
    <img src="image.jpg" alt="描述">
</picture>
```

##  已知问题与限制

### 1. 懒加载
- ⚠️ SEO影响：搜索引擎爬虫可能无法索引懒加载图片
  - **缓解**：首屏图片不使用懒加载
- ⚠️ 旧浏览器不支持IntersectionObserver
  - **缓解**：已实现降级方案（直接加载所有图片）

### 2. 暗黑模式
- ⚠️ 第三方库（如Font Awesome图标）可能未完全适配
  - **缓解**：检查并添加深色模式样式
- ⚠️ 图片颜色可能不协调
  - **缓解**：准备深色模式专用图片

### 3. WebP
- ⚠️ 旧浏览器不支持（IE、Safari < 14）
  - **缓解**：使用<picture>标签提供fallback
- ⚠️ 转换工具需要安装cwebp
  - **缓解**：提供下载安装指南

### 4. GZIP压缩
- ️ 已压缩资源（如图片）二次压缩效果有限
  - **说明**：GZIP主要对文本资源有效

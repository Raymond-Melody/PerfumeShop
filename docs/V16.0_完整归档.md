## V16 迭代历程

### V16.0 - 功能完善与体验优化

#### 一、V15遗留项收尾 (P0-P1)

**FEATURE_PASSWORD_V3 = True**
- SHA-512 + 1000轮迭代 + per-user salt
- 兼容 V1/V2 旧密码格式
- 登录时自动升级弱密码哈希
- 涉及文件: config.asp, password_utils.asp, user/login.asp, user/register.asp

**FEATURE_DAL_ENABLED = True**
- 统一数据访问层 (Data Access Layer)
- 新增文件: dal.asp, dal_products.asp, dal_users.asp, dal_orders.asp
- SafeSelect() 表存在性检查
- 慢查询日志 (DAL_SLOW_QUERY_THRESHOLD=500ms)

**FEATURE_MSOLEDBSQL = False** (已诊断,待启用)
- MSOLEDBSQL 19 驱动已安装
- 未启用原因: SQL Server实例 YOURPERFUME 仅监听 Shared Memory
- 启用前提: SQL Server配置管理器 → 启用 TCP/IP 协议 → 重启服务
- connection.asp 回退机制已加固

#### 二、API标准化 (P1)

**FEATURE_API_V1 = True**
- 统一 JSON 响应格式: {"code":200/400/403/500,"message":"...","data":{...}}
- 新增 includes/api_response.asp (API_Success/API_Error/API_Unauthorized)
- 17个API端点全部标准化:
  - 购物车: cart_add, cart_update, cart_remove, cart_clear, cart_count
  - 收藏: favorites
  - 订单: order_cancel, order_confirm
  - 地址: get_areas, get_area_name
  - 通知: notifications_sse
  - 管理: batch_operations, export_data
  - 其他: track, upload, risk_check, backup_status

**V16_API文档.md** (604行)
- 所有17个API端点完整文档
- 包含请求参数、响应格式、错误码、前端示例

#### 三、通知系统 (P2)

**FEATURE_SSE_NOTIFICATIONS = True**
- 新增 api/notifications_sse.asp (Server-Sent Events)
- 实时推送: 订单状态变更、库存预警、促销活动

**FEATURE_EMAIL_NOTIFICATIONS = True**
- 新增 includes/email_service.asp
- 集成场景: 注册欢迎、订单确认、发货通知、密码重置

#### 四、数据分析 (P2)

**FEATURE_ANALYTICS_DASHBOARD = True**
- 新增 admin/analytics/index.asp
- 销售趋势、用户增长、热销排行

#### 五、基础设施增强

**FEATURE_STRUCTURED_LOGGING = True**
- 新增 includes/logger.asp (结构化日志)
- 新增 includes/audit_utils.asp (管理员操作审计)
- 新增 admin/system/audit_logs.asp (审计日志查看)

**FEATURE_CACHE_MANAGER = True**
- 新增 includes/cache_manager.asp
- Application级缓存 (CacheGet/CacheSet, TTL支持)

**性能监控**
- 新增 includes/performance.asp (页面执行时间统计)

#### 六、PWA增强 (P2)

**FEATURE_PWA_ENHANCED = True**
- sw.js: Stale-While-Revalidate 缓存策略
- manifest.json: 更新图标和启动配置
- offline.html: 离线页面优化

#### 七、体验优化

**color-scheme 修复（重要Bug修复）**
- 根因: header.asp 中 `<meta name="color-scheme" content="light dark">` 导致浏览器在暗色模式下给表单控件注入白色文字
- 现象: 登录页、联系页输入框白色文字+白色背景 → 完全不可见
- 修复:
  - header.asp: `light dark` → `light only`
  - style.css: input/select/textarea 基础规则添加显式 color + background
  - pages.css: .auth-form input + 2处 .form-group 添加 color + background

**CSS/JS版本管理**
- 所有CSS/JS引用版本号: v15.0 → v16.2
- 12处CSS引用 + 预加载链接统一更新

**移动端响应式**
- responsive.css 移动端适配完善
- 触控友好按钮尺寸和间距

#### 八、新增工具

**api/system_diag.asp** (独立数据库诊断页)
- 独立连接测试 (不依赖connection.asp,避免Response.End终止)
- 报告: DB连接状态、TCP端口、数据表数量、Feature Flags、Session、服务器环境

**api/batch_operations.asp** (批量操作)
**api/export_data.asp** (数据导出)

#### 九、国际化预留 (P3)

**FEATURE_I18N = False** (未启用)
- includes/i18n.asp 已就绪
- locale/zh-CN.asp, locale/en-US.asp 已创建
- 框架已搭建，待翻译内容填充

## 部署指南

### 上线前检查
- [ ] AMP产品页通过Google验证
- [ ] 骨架屏在所有页面正常显示
- [ ] CDN账号已申请并配置
- [ ] CDN域名已解析
- [ ] web.config路由规则已添加
- [ ] 图片已上传至CDN

### 启用CDN步骤
1. 修改`cdn_config.asp`：
   ```vbscript
   CDN_ENABLED = True
   CDN_DOMAIN = "https://cdn.perfumeshop.com"
   ```

2. 上传静态资源到CDN：
   ```powershell
   .\ossutil64 cp -r images "oss://perfumeshop-cdn/images/"
   ```

3. 更新web.config：
   ```xml
   <rewrite>
       <rules>
           <rule name="ImageCDN" stopProcessing="true">
               <match url="^images/(.*)$" />
               <action type="Rewrite" url="https://cdn.perfumeshop.com/images/{R:1}" />
           </rule>
       </rules>
   </rewrite>
   ```

4. 清除CDN缓存并测试

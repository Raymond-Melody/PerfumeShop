##  技术架构

### 懒加载实现
```javascript
// IntersectionObserver配置
var config = {
    rootMargin: '200px 0px',
    threshold: 0.01
};

// HTML结构
<img data-src="真实图片.jpg" 
     src="占位图.svg" 
     class="lazy-placeholder">
```

### 暗黑模式实现
```javascript
// 主题优先级
1. localStorage用户选择
2. 系统偏好（prefers-color-scheme）
3. 默认亮色

// CSS变量
[data-theme="dark"] {
    --bg-color: #1A1A1A;
    --text-color: #E0E0E0;
    --product-title: #F0F0F0;
    --product-price: #FFD700;
}
```

### 手势滑动实现
```javascript
// 触摸事件
touchstart: 记录起点
touchmove: 检测水平滑动
touchend: 计算方向并切换图片

// 阈值：50px
// 防误触：diffX > diffY
```

##  核心经验

### ✅ 成功实践
1. **IntersectionObserver** - 现代浏览器懒加载标准方案
2. **GZIP压缩** - IIS配置简单，效果显著（-25%~-38%）
3. **CSS变量主题** - 暗黑模式实现优雅
4. **localStorage记忆** - 用户体验友好
5. **手势滑动** - 移动端产品浏览体验优化
6. **Null值防护** - VBScript HTMLEncode安全性提升

### ⚠️ 注意事项
1. 懒加载图片首屏不使用（影响SEO和LCP）
2. WebP需要fallback支持旧浏览器
3. 主题切换需考虑所有组件的对比度
4. GZIP对已压缩资源效果有限
5. 手势滑动需防误触（50px阈值）
6. PowerShell脚本避免中文注释

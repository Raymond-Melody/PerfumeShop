## 🎨 关键样式特性

### 移动优先CSS
```css
/* 基础样式（移动端） */
.grid { grid-template-columns: 1fr; }

/* 平板增强 */
@media (min-width: 768px) {
    .grid { grid-template-columns: repeat(2, 1fr); }
}

/* 桌面增强 */
@media (min-width: 992px) {
    .grid { grid-template-columns: repeat(4, 1fr); }
}
```

### 触摸优化
```css
.touch-target {
    min-height: 44px;
    min-width: 44px;
}

button, a, .btn {
    touch-action: manipulation;
}
```

### 安全区域（iPhone刘海）
```css
@supports (padding: max(0px)) {
    .mobile-nav {
        padding-left: max(12px, env(safe-area-inset-left));
        padding-right: max(12px, env(safe-area-inset-right));
    }
}
```

## 🎯 核心经验

### ✅ 成功实践
1. **移动优先** - 先写移动端样式，再用媒体查询增强
2. **CSS变量** - 统一设计令牌，便于主题切换
3. **组件化** - 移动端导航独立为ASP include文件
4. **触摸优化** - 所有交互元素≥44px
5. **安全区域** - iPhone刘海屏适配

### ⚠️ 注意事项
1. ASP include文件不能包含`<%@ Language %>`指令
2. 移动端图片使用`aspect-ratio`而非固定高度
3. 表单输入框font-size≥16px防止iOS自动缩放
4. 使用`env(safe-area-inset-*)`适配刘海屏
